# seaquest-coldwar
Seaquest Coldwar é uma reimaginação moderna do clássico Seaquest do Atari 2600, ambientada em um tenso cenário subaquático da Guerra Fria. O jogador comanda um submarino encarregado de resgatar mergulhadores enquanto navega por águas hostis repletas de submarinos inimigos avançados, tecnologia experimental e ameaças estratégicas.
